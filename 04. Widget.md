# Widget

Aplikasi flutter menggunakan widget sebagai element/ penyusunnya. Biasanya widget memiliki parameter untuk styling seperti `backgourndColor` yang bertipe objek `Color()`

## Color

Objek berisi warna, dapat dibuat melalui constructor `Color().fromARGB()` atau langsung akses predefined `Colors.[name]`. Biasanya digunakan untuk prop `backgroundColor`.

## MaterialApp

Merupakan root element yang dibutuhkan kebanyakan element lainnya, memiliki prop `home` untuk root page.

## Scaffold

Digunakan sebagai layout yang memiliki base style untuk childrennya. Beberapa props dapat diberikan:

- folatingActionButton: `FloatingActionButton()` floating di pojok kanan bawah
- bottomNavigator: `BottomNavigationBar(items: [BottomNavigationBarItem])` menu2 dibagian bawah
- drawer: `Drawer()` untuk sidebar ditrigger dengan hamburger button.

## AppBar

Untuk mengisi prop `appBar` dari komponen seperti `scaffold`. Merupakan navigasi bagian atas `scaffold`, berisi title, secara default memiliki tombol panah kembali jika halaman tersebut diakses dengan `navigator.push`.

## Button

### TextButton

Merupakan widget tombol, bisa diatur field `style` dengan `TextButton.styleFrom()`, memiliki constructor `.icon()`.

## Container

merupakan element seperti div pada web yang memiliki:

- `child` isi komponen
- `margin` EdgeInsets.all()
- `padding` EdgeInsets.all()
- `color`
- `height`
- `width`
- `decoration` berisi `BoxDecoration()` yang bisa mengatur
  - shadow
  - border: Border.all(width, color)
  - gradient
  - shape
- `align`: Alignment.center

## Padding

Merupakan pembungkus widget yang bisa diberikan parameter padding agar widget memiliki space dengan widget sekitarnya.

## SizedBox

merupakan widget yang sering digunakan untuk membuat jarak antara widget lain (seperti <br>), sizedbox memiliki parameter height dan width.

Selain itu, dapat digunakan untuk menampung konten. Tetapi konten akan terpotong jika melebihi ukuran dari sizedbox. Untuk itu bisa di-set `width: double.infinity`.

## Center

Digunakan untuk meletakkan komponen ditengah (secara horizontal dan vertical)

## List View

Untuk menampilkan list panjang scrollable. Agar lebih efektif (akan dirender ketika visible), gunakan builder.

```dart
ListView.builder(
  itemCount: 12,
  itemBuilder:(context, index)=> Dismissable(
    key: ValueKey(myVar[index]),
    onDismissed: (direction)=>{},
    child: ListTile(title: Text('test'))
  )
)
```

## Text

Menampilkan text, memiliki property style untuk mengatur warna dll.

## Input Element

Biasanya element ini dihubungkan dengan state, dimana akan mengubah state ketika ada event `onChange`

### TextField

Digunakan sebagai input text, berisi beberapa param `maxLength`, `keyboardType`, `decoration: InputDecoration(label: Text('title'))`. Bisa dikombinasikan dengan `TextEdittingController`.

```dart
final _titleController = TextEdittingController();
// mendapatkan text tanpa simpan di state
_titleController.Text

@override
void dispose(){
  _titleController.Dispose();
  super.dispose()
}

@override
Widget Build(){
  TextField(controller: _titleController, decoration: InputDecoration(label: Text('Title')))
}
```

### Dropdown Button

DropdownButton(children: [DropdownButtonItem(value:'sda', child: Text('label'))])

## Icon

Menampilkan icon yang disediakan flutter, icon(Icons.abc), akan ditampilkan preview jika menggunakan IDE.

## Column & Row

Berisi prop `children` yang bisa diletakkan komponen secara `vertikal (column)` dan `horizontal (row)`. Secara default mengambil space penuh, dapat diubah dengan `mainAxisSize`. Disini konten children akan mengambil space sekecil mungkin dan jika parent nya merupakan center, akan membuatnya ketengah. Kita dapat mengatur nilai `mainAxisAlignment: MainAxisAlignment.spaceAround`

## Spacer

Digunakan untuk mengambil space sebanyak2nya diantara child Column atau Row.

## Opacity

Digunakan untuk mengubah `transparansi` tampilan komponen child dalamnya. Tetapi cukup `performa intensif`, untuk komponen yang bisa diatur `color`, kita juga bisa mengatur transparansi dalamnya.

## Stack

membuat seluruh childrennya tumpuk satu sama lain, biasanya dikombinasikan dengan:

- `Positioned()` untuk mengatur posisi child seperti absolute css.
- `Align()` untuk mengatur posisi child relatif parent

## Utility

Flutter memiliki beberapa utility untuk menampilkan widget

- `showDatePicker()` tampilin datepicker
- `showModalBottomSheet()` tampilin modal/drawer bawah, bisa full screen dengan param `isScrollControlled: true`, bisa juga diberikan `useSafeArea: true` agar memberikan ruang top system bar.
- `showDialog()` tampilin dialog (error, sukses, )

```dart
AlertDialog(title: Text('title'), content: Text(''), actions: [])
```

Utility class biasanya menampilkan modal atau drawer diatas tampilan utama. Untuk menutupnya, dapat menggunakan `Navigator.pop(context)`

## Expanded

Digunakan untuk membungkus komponen yang tidak memenuhi lebar, agar komponen didalamnya memakan seluruh space yang ada. Expanded hanya boleh dipakai sebagai children dari:
`Row, Column, atau Flex`

## SnackBar

Sebuah pesan kecil dibawah

```dart
ScaffoldMessenger.of(context).clearSnackBar(); //remove previous snackbar
ScaffoldMessenger.of(context).showSnackBar(
  SnackBar(
    duration: const Duration(seconds: 4),
    content: Text('expense deleted')
    action: SnackBarAction(label: 'Undo', onPressed: (){}, )
  )
);
```
