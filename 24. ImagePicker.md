# Image Picker

Merupakan 3rd party library yang digunakan untuk input image baik dari kamera, maupun file.

## Penggunaan

```dart
void _takePicture()async{
    final imagePicker = ImagePicker();
    final pickedImage = await imagePicker.pickImage(source: ImageSource.camera, maxWidth: 600)

    _selectedImage = File(pickedImage.path)
}

@override
Widget build(){
    return Image.file(_selectedImage, fit: BoxFit.cover)
}
```

Kita juga bisa menggunakan android virtual device untuk menangkap kamera.
