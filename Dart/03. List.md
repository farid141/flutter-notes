# List

```dart
// Penulisan tipe list
List<String>

// Pembuatan list langsung pakai []
['asd', 'dsa', 'sad'] 
```

## If statement dalam List

```dart
// Single Element
[
  Text("Selalu tampil"),
  if (isLoggedIn)
    Text("Hanya tampil kalau login"),
],

// Memasukkan multiple element (menggunakan spread operator ...[])
[
  Text("Selalu tampil"),
  if (isLoggedIn) ...[
    Text("Hello"),
    Text("World"),
  ],
]
```
