## Dynamic VS Object

Kedua tipe ini memberikan kebebasan untuk menampung data yang tidak perlu diketahui tipenya.

Namun dynamic bersifat lebih flexible, karena variabelnya bersifat relevan ketika digunakan sebagai param untuk tipe data apapun. Perlu diingat, tipe dynamic dapat menghasilkan `runtime error` karena tidak diperiksa saat `compile time`.

Sedangkan Object lebih strict dibanding dynamic, karena akan diperiksa saat `compile time` biasanya akan nampak error dari intelisense editor.
