# Media Query

Digunakan untuk menddapatkan nilai system berjalan

dapatkan parameter env system berjalan
`final isDarkmode =  == Brightness.dark;`

- `MediaQuery.of(context).platformBrightness` system theme
- `MediaQuery.of(context).size.[width/height]` current dimension size, bisa digunakan if-else untuk responsive, bisa juga pake `<LayoutBuilder>` untuk mendapatkan constraints (batasan tinggi/lebar) yang bisa ditempati komponen. 
- `MediaQuery.of(context).viewInsets.bottom` untuk mengambil ukuran yang digunakan keyboard (muncul dari bawah) agar bisa menentukan ukuran padding (semisal ternyata bottom padding cuma sedikit, sehingga waktu open keyboard tidak bisa menggser ke bawah (kita harus memberikan ruang padding untuk keyboard))

Ketika parameter berubah, akan terjadi rerendering `build` method
