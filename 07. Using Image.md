# Using Image

Kita bisa menggunakan image dengan class builtin `Image.assets('img_path')`. Sebelum itu, pastikan tambahkan file assets pada pubspec.yml

```yml
flutter:
    assets:
        - assets/logos/
        - assets/icons/
        - assets/images/
    fonts:
        - family: Poppins
          fonts:
            - asset: path
              weight: 300
```

Gunakan slash pada folder untuk menginclude semua file assets dalam folder tersebut.
